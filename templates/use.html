Welcome to Libra API

Follow the steps below to interact with the Libra API:

Step 1: Start the Server

go run main.go

Step 2: Make a Request

Use the following curl command to send a request:

curl "http://localhost:10000/libra?q=What+is+AI?"

Step 3: Sample Response

{ "response": "AI stands for Artificial Intelligence..." }

Step 4: Enable Streaming

To receive streaming responses, add the stream=true parameter:

curl "http://localhost:10000/libra?q=Explain+quantum+computing&stream=true"
